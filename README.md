# Gestor de Eventos – Sistema Web ⋆˙⟡

Sistema web para la gestión integral de eventos, orientado a salones y organizadores que necesitan centralizar la información, controlar estados y optimizar la comunicación entre clientes, empleados y administradores.

El proyecto fue desarrollado como **Trabajo Final Integrador** y simula un escenario real de uso.

# Descripción general

⤷ La plataforma permite que los clientes se registren, creen eventos y los envíen a revisión, mientras que los empleados pueden aprobarlos, rechazarlos o gestionarlos según su estado. El sistema incorpora control de roles, notificaciones automáticas y herramientas administrativas para el seguimiento y la toma de decisiones.

## Funcionalidades principales
**✶ Cliente**

- Registro e inicio de sesión
- Creación de eventos con selección de fecha y elementos
- Visualización de disponibilidad
- Presupuesto automático
- Envío de eventos a revisión
- Seguimiento del estado del evento
- Notificaciones por correo electrónico
- Recuperación de contraseña por mail

**✶ Empleado**
- Gestión y supervisión de eventos
- Aprobación, rechazo y cambio de estado
- Consulta de disponibilidad
- Gestión del catálogo de elementos
- Envío de notificaciones automáticas a clientes

**✶ Administrador**
- Gestión de empleados

- Auditoría completa del sistema

- Seguimiento de acciones por usuario

- Paneles con métricas y estadísticas (KPIs)

**✶ Invitado**

- Acceso mediante enlace público

- Visualización de invitación del evento

- Contador regresivo

- Mapa de ubicación del evento

## Extras

- Bot informativo con respuestas predefinidas ッ
- Sistema de auditoría y trazabilidad
- Control de estados: pendiente, aprobado, rechazado, cancelado y completado
